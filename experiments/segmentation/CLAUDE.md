# Segmentation Experiment

## Goal
Separate collective bargaining agreements into their Articles or sections based on
headers and a two-tier hierarchy. Input is one large document text corpus (built
from OCR pages). Output is a JSON object with subsection records shaped like:

`{"parent": "...", "title": "...", "text": "..."}`

## KPI
Compare methods by overlap in identified section text.

Primary comparison metric:
- Mean section overlap (token Jaccard) by `(parent, title)` key.

Secondary comparison metric:
- Coverage IoU from extracted character spans across the document.

## Methods
- `llm_segment`
  - First pass reads the first 10-25 pages to infer hierarchy conventions.
  - Second pass processes overlapping chunks and emits subsection segments with
    parent section labels.
- `llm_segment_v2`
  - First pass reads the first 10-25 pages to infer hierarchy conventions.
  - Generates deterministic boundary candidates from regex/line heuristics.
  - Verifies and normalizes candidates with LLM.
  - Builds subsection spans deterministically from verified boundaries.
  - Falls back to coarse chunk extraction if no segments survive (optional).

## Model Variants
- `gpt5mini` -> `openai/gpt-5-mini`
- `gemini25flash` -> `google/gemini-2.5-flash`
- `claudehaiku45` -> `anthropic/claude-haiku-4.5`

Example runs:
- `python experiments/segmentation/run_experiment.py --doc-id 864 --variant gpt5mini --methods llm_segment,llm_segment_v2`
- `python experiments/segmentation/run_experiment.py --doc-id 864 --variant gemini25flash --methods llm_segment,llm_segment_v2`
- `python experiments/segmentation/run_experiment.py --doc-id 864 --variant claudehaiku45 --methods llm_segment,llm_segment_v2`
- `python experiments/segmentation/run_experiment.py --doc-id 864 --all-variants --methods llm_segment,llm_segment_v2`
- `python experiments/segmentation/run_experiment.py --doc-id 864 --methods llm_segment_v2`

## Outputs
- `output/<document_id>/<method>__<run_tag>.json`
- `results.csv` (run rows + overlap rows)
- `report.md` (generated by `analyze_results.py`)
